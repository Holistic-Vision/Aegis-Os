# Courses (MVP)

Comparatif “en temps réel” des prix par magasins **nécessite une API externe** (non disponible en pur GitHub Pages).  
Ici : une base de travail + listes adaptables (selon profil).

## Panier de base (7 jours)
### Protéines
- Œufs
- Poulet / dinde
- Lentilles / pois chiches
- Skyr / fromage blanc (option)

### Féculents
- Riz
- Pâtes complètes
- Flocons d’avoine
- Pommes de terre

### Légumes
- Mélanges surgelés
- Salade
- Oignons
- Tomates

### Fruits
- Bananes
- Pommes
- Agrumes

### Divers
- Huile d’olive
- Épices (curcuma, paprika, poivre)
- Sel iodé
- Moutarde

## Astuce anti-allergies/intolérances
- Lactose : prendre “sans lactose” ou alternatives soja/amande.
- Gluten : riz/pommes de terre/quinoa plutôt que pâtes/pain.

### Source utile (produits)
- Open Food Facts : https://world.openfoodfacts.org/
